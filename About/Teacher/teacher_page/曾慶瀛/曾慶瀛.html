曾慶瀛博士（名譽教授） Chin-Yin Tseng, Ph. D.
台灣大學食品科技研究所博士
中興大學食品科學研究所碩士
中興大學食品科學系學士

教師研究成果

國立嘉義農專食品工業科助教 (1975-1977)

國立嘉義農專食品工業科講師 (1982-1990)

國立嘉義農專食品工業科副教授兼工廠主任 (1990-1994)

國立嘉義技術學院食品科學系教授兼系主任 (1994-2000)

國立嘉義大學食品科學系教授兼研發處處長 (2000-2002 ； 2004-2008)

中州技術學院保健營養系講座教授兼副校長 (2008-2011)

中州技術學院代理校長 (2010/08-09)

中州科技大學保健食品系 ( 含碩士班 ) 講座教授兼副校長 (2011-2012)

中州科技大學保健食品系 ( 含碩士班 ) 講座教授兼代理校長 (2012/04-07)

中州科技大學保健食品系 ( 含碩士班 ) 校長兼講座教授 (2012/08-2017/07)

一、建置食品加工廠，負責校慶產品生產前後達 10年。

二、爭取獲政府單位補助款增購設備：殺菌軟袋機、 IQF、檢驗分析儀器等多項設備

三、建置食品系精密儀器中心、研究室實驗桌、教授休息室、演講廳、編撰花生加工專輯。

四、推薦系上多位教師至台灣大學、中興大學進修碩士、博士學位。

五、多年受外交部聘任擔任食品加工專家並赴南太平洋島國 (斐濟、密克羅尼西、索羅門群島、庫克群島、吉里巴斯群島、萬那杜 )及中美洲 (哥斯大黎加、宏都拉斯、尼加拉瓜、貝里斯 )授課

六、引進南太平洋島國政府官員至嘉義大學受訓。

七、多次參與國際研討會

八、受亞太科技協會多次遴選赴日本考察日本教育制度、食品專業及生質能源專業。

九、指導國立嘉義大學食品科學研究所碩士班學生多達 31位

十、任職嘉義大學期間執行教育部、行政院國家科會委員會、農業委員會、各鄉鎮農會、企業等計畫共 90多件，研究經費多達數千萬

十一、任教期間發表期刊論文共計 24篇，學術研討會論文共計 57篇。

十二、曾擔任台灣大學、海洋大學、中興大學、屏東科大、輔仁大學、大葉大學等大學之博、碩士學位口試委員。
